# 적정가 산정 방법론

## 기본 원칙

### 왜 12M Forward EPS인가

다음 세 가지 EPS 기준 중 12M Forward를 사용하는 이유:

| 기준 | 장점 | 단점 |
|------|------|------|
| TTM (과거 4분기) | 실제값, 검증 가능 | 이익 모멘텀 구간에서 무용지물 (PER 30배가 8배일 수 있음) |
| 단년 컨센서스 (2026E) | 명확함 | 사이클 정점에 잡힐 위험 |
| **12M Forward (향후 4분기 합산)** | **균형, 글로벌 IB 표준** | **컨센서스 의존** |

이익 모멘텀이 강한 현재(2026년) 한국 증시에서는 12M Forward가 가장 적합하다.

### 산정 공식

**일반 흑자 종목**:
```
적정가 = 12M Forward EPS × Target PER
```

**지주사 (NAV 방식)**:
```
적정가 = 보유 자산 NAV 합산 / 발행주식수 × (1 − 할인율)
```

**적자 / 소이익 성장주 (PSR 방식)**:
```
적정가 = 주당 매출 × Target PSR
```

## 4단계 시나리오 정의

각 종목별로 4가지 시나리오를 산출한다:

### 보수 (Conservative)
- **가정**: 현재 사이클이 정점에 가깝다. 이익 둔화 시작.
- **PER 적용**: 과거 5~10년 PER 밴드의 **하단** 또는 사이클 정점 평균
- **NAV 할인율**: 평균 할인율의 **상단** (최대 할인)
- **PSR 적용**: 과거 평균 PSR의 **하단**

### 기준 (Base)
- **가정**: 컨센서스대로 진행. 시장 평균 멀티플 유지.
- **PER 적용**: 컨센서스 Target PER 또는 글로벌 Peer 중위값
- **NAV 할인율**: 한국 지주사 평균 할인율 (30~40%)
- **PSR 적용**: 글로벌 동종 업계 평균

### 낙관 (Optimistic)
- **가정**: 사이클 강세 지속, 마진 확대.
- **PER 적용**: 과거 PER 밴드의 **상단**
- **NAV 할인율**: 평균 할인율의 **하단** (최소 할인)
- **PSR 적용**: 동종 업계 상위 분위

### AI 재평가 (Re-rating)
- **가정**: 엔비디아·TSMC처럼 산업 구조가 재정의됨. AI 시대 핵심 종목으로 재평가.
- **PER 적용**: 글로벌 AI 관련주 평균 또는 그 이상
- **NAV 할인율**: 매우 작은 할인 (10~20%)
- **PSR 적용**: 성장주 평균
- ⚠️ **주의**: 단기 목표가가 아닌 구조적 변화 시나리오

## 종목별 산정 사례

### 사례 1: 일반 흑자 종목 (HD현대일렉트릭)

```
12M Forward EPS = 31,000원 (2026F 지배순이익 9,990억원 / 발행주식)
보수 PER = 18배 → 558,000원
기준 PER = 27배 → 837,000원
낙관 PER = 35배 → 1,085,000원
재평가 PER = 45배 → 1,395,000원
```

근거: 유안타 2026F PER 26.9배. 키움 27F EPS × Target PER 30배 = 106만원.

### 사례 2: 지주사 (SK스퀘어)

```
NAV 추정 = 주당 약 200만원 (SK하이닉스 지분 20.1% + ICT 자회사)
보수 = 200만 × (1 − 0.40) = 1,200,000원
기준 = 200만 × (1 − 0.30) = 1,400,000원
낙관 = 200만 × (1 − 0.20) = 1,600,000원
재평가 = 200만 × (1 − 0.10) = 1,800,000원
```

근거: 한국 지주사 통상 30~50% 할인. AI 시대 모회사 가치 재평가 시 할인 축소.

### 사례 3: 적자 성장주 (두산로보틱스)

```
주당 매출 ≈ 1,300원 (매출 800억 / 발행주식)
보수 PSR = 8배 → 10,400원
기준 PSR = 15배 → 19,500원
낙관 PSR = 25배 → 32,500원
재평가 PSR = 40배 → 52,000원
```

근거: 글로벌 협동로봇 Peer 평균 PSR 약 15배. 휴머노이드 비전 반영 시 확장.

## 가정의 한계와 정직한 표현

### 사이클 함정 경고

**중요한 통찰**: 시클리컬 종목은 다음 패턴을 보인다.
- 사이클 정점: 이익 최대 → PER 최저 → "싸 보임"
- 사이클 저점: 이익 최저 → PER 최고 → "비싸 보임"

**대응**:
- 메모리(삼성·하이닉스), 중공업(두산에너빌리티), 변압기 종목은 사이클 위치를 함께 표시
- 보수 시나리오는 "사이클 정점일 경우의 PER" 적용

### 컨센서스 자체의 한계

증권사 컨센서스는:
- 같은 회사 분석 + 군중 심리로 한 방향 쏠림 가능
- 평균 +18%의 EPS 서프라이즈 (2026 1Q 기준) → 컨센서스가 낮을 가능성도 있음
- 분기 실적 후 빠르게 상향/하향됨

**대응**: 컨센서스를 "기준 시나리오"로만 사용. 보수와 낙관에서 ±20~30% 폭으로 펼침.

### 환율과 거시 변수

원/달러 환율이 10% 변동하면 수출주 EPS도 큰 폭 변동:
- 노무라: "원화 10% 강세 시 SK하이닉스 달러 매출 +9%, 영업이익 +14% 상향"
- 메모: 환율 가정도 시나리오의 일부. 1300원 vs 1400원 가정에서 결과가 다름.

## 컨센서스 vs 본 추정 교차 검증

산출 후 항상 다음을 비교한다:

```
종목별 차이 = (본 추정 기준가 − 컨센서스 평균 목표가) / 컨센서스 평균 목표가
```

- 차이 ≤ 10%: 신뢰도 ↑
- 10% < 차이 ≤ 30%: 가정 점검 (EPS 또는 PER 어느 쪽이 다른지)
- 차이 > 30%: 본 추정 또는 컨센서스 중 하나가 강한 견해. 메모 필수.

특히 **"컨센서스 목표가 < 현재가" 종목은 시장이 이미 과열로 보고 있다는 신호**로 명시한다 (예: LS ELECTRIC, 대한전선, 레인보우로보틱스).

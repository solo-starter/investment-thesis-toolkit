# 데이터 수집 검색 쿼리 가이드

## 목적

각 종목에 대해 효율적으로 필요한 데이터를 모으기 위한 검색 쿼리 템플릿. 쿼리는 짧고 구체적으로 (1~6 단어).

## 종목당 표준 검색 쿼리 (3~4개)

### 쿼리 1: 기본 시장 정보
```
[종목명] [종목코드] 주가 시가총액 PER 2026
```
**예시**: `삼성전자 005930 주가 시가총액 PER 2026`

**얻는 정보**: 현재가, 시가총액, 52주 최고/최저, TTM EPS, 배당수익률, 12개월 목표주가

**추천 출처**: Investing.com, TradingView, FnGuide

### 쿼리 2: 적정가 산정용 컨센서스
```
[종목명] 2026 영업이익 컨센서스 EPS 목표주가
```
**예시**: `SK하이닉스 2026 영업이익 컨센서스 EPS 목표주가`

**얻는 정보**: 2026/2027 영업이익·순이익 전망, Target PER, 12M Forward EPS

**추천 출처**: 증권사 리포트, 경제 뉴스

### 쿼리 3: 차기 실적 발표일
```
[종목명] [종목코드] 차기 실적 발표일 2026 [분기]
```
**예시**: `삼성전자 005930 차기 실적 발표일 2026 2분기`

**얻는 정보**: 정확한 발표 예정 날짜

**추천 출처**: Investing.com (실적 캘린더), 회사 IR

### 쿼리 4 (선택): 최근 실적/뉴스
```
[종목명] [최근 분기] 실적 영업이익 컨센서스
```
**예시**: `SK하이닉스 1Q26 실적 영업이익 컨센서스`

필요한 경우만 사용. 컨센서스 변화의 맥락 이해용.

## 효율화 전략

### 그룹 검색 활용
여러 종목을 한 쿼리로 검색 (특히 같은 섹터):
```
효성중공업 LS HD현대일렉트릭 대한전선 2026 영업이익 컨센서스
SK하이닉스 SK스퀘어 2026 컨센서스 PER
두산로보틱스 두산에너빌리티 레인보우로보틱스 2026 매출 PER
```

### 출처 우선순위
1. **1순위**: Investing.com (실시간 주가 + 실적 일정)
2. **2순위**: TradingView (PER, 베타, 컨센서스)
3. **3순위**: FnGuide / Company Guide (재무 상세)
4. **4순위**: 증권사 리포트 PDF (목표주가, 시나리오)
5. **5순위**: 경제 뉴스 (Hankyung, NewsSpace, 머니투데이)

### 쿼리 작성 주의사항
- **현재 연도 사용**: "2026" (현재가 2026년이면). "2025"로 검색하면 stale 결과
- **종목명 정확히**: "SK하이닉스" (한글), "000660" (숫자) - 둘 다 시도
- **인용부호, site:, - 기호 사용 금지**
- **6단어 이내**

## 종목별 특이사항

### 삼성전자 (005930)
- 분기 영업이익 단위가 매우 큼 (조 단위)
- HBM 매출, MX, DS, 하만 부문별 분리 검색 가능

### SK하이닉스 (000660)
- HBM 가이던스가 핵심
- 12M Forward PER이 가장 자주 인용됨 (5.2배 등)

### SK스퀘어 (402340)
- 컨센서스 목표주가 < 현재가 (자주 발생)
- NAV 산정 시 SK하이닉스 시가총액 × 20.1% 필수

### LS (006260)
- LS전선 (해저케이블) 검색이 더 정보 풍부
- 자회사 LS일렉트릭, LS MnM 통합 NAV 필요

### 두산에너빌리티 (034020)
- 시총 81조원, 그러나 EPS 작아 PER 매우 높게 보임
- 가스터빈/SMR 신사업 기대치가 핵심

### 두산로보틱스 (454910)
- 적자 종목. PER 무의미
- 매출 추이가 핵심 지표

### 레인보우로보틱스 (277810)
- 코스닥 상장
- 삼성전자 자회사 편입 후 멀티플 매우 높음

## 검색 후 데이터 검증

수집한 데이터에 대해 다음 검증을 한다:

1. **현재가 일치성**: 여러 출처에서 가격이 일치하는가?
2. **시가총액 = 현재가 × 발행주식수**: 단순 산수 검증
3. **EPS 일관성**: TTM EPS × PER ≈ 현재가 인가?
4. **목표주가 범위**: 최저~최고 목표주가가 합리적 범위인가?
5. **실적 발표일**: 분기 종료 후 통상 3~6주 내인가?

불일치 시 가장 최근 출처 우선. 의심스러우면 추가 검색.

## 시간이 부족할 때의 우선순위

11개 종목 × 3 쿼리 = 33회 검색은 부담스러울 수 있다. 우선순위:

1. **필수 (모든 종목)**: 쿼리 1 (기본 정보)
2. **중요 (적정가 분석에 핵심)**: 쿼리 2 (컨센서스)
3. **선택 (다른 출처에서 이미 확인된 경우 생략 가능)**: 쿼리 3, 4

그룹 검색으로 효율화하면 11개 종목을 6~8회 검색으로 처리 가능.

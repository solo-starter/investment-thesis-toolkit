# /적정가산정 [종목명]

종합 적정가치 평가 명령어. 한국 종목에 대해 4단계 시나리오로 적정가를 산정한다.

## 트리거

다음 중 하나가 사용자 메시지에 있으면 이 명령어를 실행한다:

**슬래시 명령어 형식**:
- `/적정가산정 [종목명]`
- `/적정가산정 [종목코드]`
- `/fairvalue [종목명]` (영문 별칭)

**자연어 트리거**:
- "[종목명] 적정가 산정해줘"
- "[종목명] 가치평가"
- "[종목명] 본질가치"
- "[종목명] 얼마가 적정해?"
- "[종목명] 가격 평가"

## 작업 흐름

### Step 1: 종목 식별 및 분류
사용자가 입력한 종목명/코드로 다음을 판단한다:
- 한국 11개 핵심 포트폴리오 종목인가? → `references/per_bands.md`의 사전 정의 PER 밴드 사용
- 신규 종목인가? → `references/per_bands.md`의 "새 종목 추가 시 PER 밴드 설정 방법" 적용

산정 방식 결정:
- 일반 흑자 종목 → **PER × 12M Forward EPS**
- 지주사 (SK스퀘어, LS 같은) → **NAV × (1 − 할인율)**
- 적자/소이익 성장주 (두산로보틱스, 레인보우로보틱스 같은) → **PSR × 매출**

### Step 2: 데이터 수집
`references/data_collection.md`의 표준 검색 쿼리 3개 실행:
1. 기본 시장 정보 (현재가, 시가총액, TTM EPS)
2. 컨센서스 데이터 (12M Forward EPS, 2026/2027 영업이익)
3. 컨센서스 목표주가 평균 + 매수/매도 애널리스트 수

### Step 3: 4단계 시나리오 적정가 산출
`references/valuation_method.md`의 방법론 적용:

**일반 흑자 종목**:
```
적정가_보수 = 12M Fwd EPS × PER_보수 (밴드 하단)
적정가_기준 = 12M Fwd EPS × PER_기준 (컨센 Target)
적정가_낙관 = 12M Fwd EPS × PER_낙관 (밴드 상단)
적정가_재평가 = 12M Fwd EPS × PER_재평가 (글로벌 Peer 평균+)
```

**지주사**:
```
적정가_시나리오 = NAV × (1 − 할인율_시나리오)
```

**적자/성장주**:
```
적정가_시나리오 = 주당매출 × PSR_시나리오
```

### Step 4: 컨센서스 교차 검증
```
차이 = (본 추정 기준가 − 컨센서스 평균 목표가) / 컨센서스 평균 목표가
```
- ≤ 10%: 신뢰도 ↑
- 10~30%: 가정 점검 (EPS 또는 PER 어느 쪽이 다른지 명시)
- > 30%: 강한 견해 차이. 메모 필수

### Step 5: 결과 출력

다음 형식의 표로 응답한다:

```
📊 [종목명] (종목코드) 적정가 분석

기본 정보
- 현재가: XXX,XXX원
- 시가총액: XX조원
- 산정 방식: PER × EPS / NAV 할인 / PSR × Sales
- 12M Fwd EPS: XX,XXX원

4단계 시나리오 적정가
| 시나리오 | PER (또는 할인율/PSR) | 적정가 | 상승여력 |
| 보수    | 8x                  | 280,000  | +4%     |
| 기준    | 11x                 | 385,000  | +43% ⭐ |
| 낙관    | 14x                 | 490,000  | +83%    |
| AI 재평가 | 18x               | 630,000  | +135%   |

컨센서스 비교
- 컨센서스 평균 목표가: XXX,XXX원
- 본 추정(기준) vs 컨센서스: +XX% 차이
- 신뢰도: ↑ / 가정 점검 필요 / 강한 견해 차이

투자 판단 가이드
- 매수권: 현재가 < 보수 적정가 → 안전 마진 ✅
- 합리적: 현재가 < 기준 적정가 → 매수 검토 가능
- 신중: 현재가 > 낙관 적정가 → 신규 매수 자제

⚠️ 주의사항
- [종목 특이 리스크]
- 모든 적정가는 가정의 함수입니다 (EPS ±10% → 적정가 ±10%)
- 본 추정은 투자 권유가 아닙니다
```

### Step 6: 후속 작업 제안 (선택)
사용자가 추가로 원할 수 있는 작업:
- "분기별 히스토리 시트에 추가할까요?"
- "차기 실적 발표일까지 어떤 점을 봐야 하나요?" → `/실적프리뷰` 자동 제안

## ETF 처리

ETF가 입력되면 `/ETF분석` 명령어로 자동 라우팅한다.

## 사용 예시

**예시 1: 일반 종목**
```
사용자: /적정가산정 삼성전자
또는: 삼성전자 적정가 산정해줘
```

**예시 2: 지주사**
```
사용자: /적정가산정 SK스퀘어
→ NAV 방식 자동 적용 (SK하이닉스 지분 20.1% + ICT 자회사)
```

**예시 3: 적자 성장주**
```
사용자: /적정가산정 두산로보틱스
→ PSR 방식 자동 적용 (적자 종목)
```

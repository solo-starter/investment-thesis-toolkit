# /실적프리뷰 [종목명]

차기 실적 발표 전 시나리오 분석을 통해 발표 후 가능한 주가 시나리오를 미리 점검한다.

## 트리거

**슬래시 명령어 형식**:
- `/실적프리뷰 [종목명]`
- `/earnings-preview [종목명]` (영문 별칭)

**자연어 트리거**:
- "[종목명] 다음 실적 어떻게 예상돼?"
- "[종목명] 차기 실적 시나리오"
- "[종목명] 실적 발표 전 점검"
- "[종목명] 실적 어떨 것 같아?"

## 작업 흐름

### Step 1: 차기 실적 일정 확인
- 발표 예정일 (캘린더에 등록된 날짜 또는 검색)
- D-day 표시
- 분기 (예: 2026 2Q)

### Step 2: 컨센서스 EPS 분포 수집
검색 쿼리: `[종목명] [차기 분기] 컨센서스 영업이익 EPS 전망 증권사`

수집 항목:
- 컨센서스 EPS 평균
- 컨센서스 EPS 최저/최고 (분포)
- 컨센서스 매출 평균
- 컨센서스 영업이익 평균
- 분포의 표준편차 (의견 일치도)

### Step 3: 3가지 실적 시나리오 가정

**Scenario A: 미스 (컨센 대비 -10%)**
- 가정: 매출/마진 둔화
- 예상 EPS, 영업이익
- 발표 후 적정가 변화 예측

**Scenario B: 컨센 부합 (Base)**
- 가정: 컨센서스 그대로
- 현재 적정가 유지

**Scenario C: 서프라이즈 (컨센 대비 +10%)**
- 가정: 마진 개선 또는 매출 호조
- 예상 EPS, 영업이익
- 발표 후 적정가 변화 예측

### Step 4: 발표 후 주가 시나리오

각 시나리오별로:
- 즉각적 주가 반응 예상 범위
- 컨센서스 상향/하향 가능성
- 신규 적정가 범위

### Step 5: 핵심 관전 포인트 추출

종목별 특이 점검 항목:
- **삼성전자**: HBM 매출, 파운드리 적자/흑자, MX/DS 부문별 실적
- **SK하이닉스**: HBM 가이던스, DRAM/NAND 가격 추이, AI 인프라 코멘트
- **효성중공업**: 미국 변압기 영업이익률 (35% 유지?), 신규 수주
- **HD현대일렉트릭**: 북미 매출 비중, 데이터센터향 변압기 수주
- **두산에너빌리티**: 가스터빈 수주, SMR 진척, 두산밥캣 실적
- **LS ELECTRIC**: 미국 빅테크 후속 수주, 영업이익률 추이
- **두산로보틱스**: 적자 폭 축소 여부, 신규 협동로봇 수주
- **레인보우로보틱스**: 매출 성장률, 삼성 시너지

### Step 6: 결과 출력

```
🔍 [종목명] 차기 실적 프리뷰

📅 발표 일정
- 예정일: YYYY-MM-DD (D-XX)
- 분기: YYYY NQ
- 발표 형식: 잠정 / 확정

📊 컨센서스 분포
- EPS 평균: XX,XXX원 (최저 YY,YYY ~ 최고 ZZ,ZZZ)
- 영업이익 평균: XX조원
- 매출 평균: XX조원
- 의견 일치도: 높음/중간/낮음

🎲 3가지 시나리오

| 시나리오 | 가정 | 예상 EPS | 예상 적정가 | 주가 반응 |
| Miss(-10%) | 마진 둔화 | XX,XXX | YYY,YYY | -X~Y% |
| Base | 컨센 부합 | XX,XXX | YYY,YYY | -X~+Y% |
| Beat(+10%) | 마진 개선 | XX,XXX | YYY,YYY | +X~Y% |

🎯 핵심 관전 포인트
1. [종목 특이 항목 1]
2. [종목 특이 항목 2]
3. [종목 특이 항목 3]

⚠️ 주의 신호
- [있는 경우만 - 예: 컨센 분산 큼 → 변동성 클 수 있음]

💡 발표 직전 행동 가이드
- D-1: 캘린더 알림 활성화 ✅ (이미 등록됨)
- D-day: 발표 직후 가이던스 확인
- D+1~7: 컨센서스 재산정 후 /실적분석 실행
```

## 사용 예시

```
/실적프리뷰 효성중공업
→ 7/24 발표 (D-75)
→ 컨센 영업이익 2,800~3,500억원
→ 미국 매출 이연분 반영 시 Beat 가능
→ 핵심: 미국 영업이익률 35% 유지 여부
```

## 한계

- 컨센서스 분포는 시간이 지남에 따라 변한다. 발표 D-1 ~ D-7 사이에 한 번 더 실행하는 것이 정확하다.
- 가이던스 변경이나 사전 워닝(pre-announcement)은 예측 불가능하다.
- 한국 시장은 실적 후 단기 변동성이 글로벌 평균보다 크다 (개인 비중 높음).

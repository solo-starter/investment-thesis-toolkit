# /ETF분석 [ETF명 또는 코드]

ETF의 적정가치를 편입 종목 가중평균으로 산출하는 명령어. (Anthropic 원본 플러그인에 없는 우리만의 명령어)

## 트리거

**슬래시 명령어 형식**:
- `/ETF분석 [ETF명]`
- `/ETF분석 [ETF코드]`
- `/etf [ETF명]` (영문 별칭)

**자연어 트리거**:
- "[ETF명] 적정가 분석"
- "[ETF명] 어때?"
- "[ETF명] 비싸 싸?"
- "[ETF명] 평가해줘"

## 작업 흐름

### Step 1: ETF 기본 정보 수집

검색 쿼리: `[ETF명] 편입종목 비중 운용보수 NAV`

수집 항목:
- ETF 정식 명칭, 운용사
- 추적 지수 / 테마
- 운용보수 (TER)
- 순자산총액 (AUM)
- 현재가 vs iNAV 괴리율
- 분배금 정책

### Step 2: 편입 종목 상위 10개 추출

검색 쿼리: `[ETF명] 구성종목 비중 상위`

수집 항목:
- 종목명, 종목코드
- 편입 비중 (%)
- 섹터 분류

**상위 10개 종목이 통상 ETF 가치의 60~80% 결정**한다.

### Step 3: 편입 종목별 적정가 산정

상위 10개 종목 각각에 대해 `/적정가산정` 로직 적용:
- 각 종목의 4단계 시나리오 적정가 (보수/기준/낙관/재평가)
- 각 종목의 현재가
- 각 종목의 상승여력 (%)

**효율화**: 우리 포트폴리오 11개 종목과 겹치는 경우 기존 데이터 재사용

### Step 4: ETF 적정가 가중평균 산출

```
ETF 적정가 (시나리오별) = Σ (편입 종목 적정가 × 편입 비중)
```

**예시 계산**:
```
KODEX AI반도체 ETF
- 삼성전자 35% × 기준 적정가 38.5만원 = 13.475만원
- SK하이닉스 25% × 기준 적정가 245만원 = 61.25만원 (주가 차이 정규화 필요)
...
```

⚠️ **주의**: ETF 가격은 종목 가격의 단순 합이 아니라 추적 지수의 단위에 비례한다. 따라서 다음 방식 사용:

```
ETF 적정 멀티플 = Σ (편입 종목 (적정가/현재가) × 편입 비중)
ETF 적정가 = ETF 현재가 × 적정 멀티플
```

### Step 5: iNAV 괴리 분석
- ETF 시장가 vs iNAV (실시간 추정 순자산가치) 비교
- 괴리율이 1% 이상이면 차익거래 기회 또는 유동성 문제

### Step 6: 추적 오차 점검
- 추적 지수 대비 ETF 수익률 차이
- 운용보수 + 추적 오차 = 실질 비용

### Step 7: 결과 출력


### Step 8: 후속 작업 제안

- "편입 종목 중 [X 종목]만 직접 매수하는 게 어떤가요?"
- "이 ETF 대신 다른 ETF를 검토해볼까요?"
- "/논리점검 [편입 핵심 종목]"

## 사용 예시

**예시 1: 테마 ETF**
```
/ETF분석 KODEX AI반도체

→ 상위 10종목 자동 분석 (포트폴리오 종목 4개 포함 → 데이터 재사용)
→ 가중평균 기준 적정가: 41,500원 vs 현재 38,200원
→ 상승여력 +8.6%
→ 가장 큰 기여: SK하이닉스 (비중 25%, 상승여력 49%)
```

**예시 2: 섹터 ETF**
```
/ETF분석 TIGER 반도체

→ 우리 포트폴리오 5개 종목 포함
→ 기준 적정가 +12% 상승여력
→ 직접 매수 vs ETF: 운용보수 0.49% 차이로 직접 매수 약간 유리
```

**예시 3: 신규 ETF**
```
/ETF분석 [새 AI 로봇 ETF]

→ 상위 종목 분석 후 우리 포트폴리오와 70% 겹침 안내
→ "직접 매수 종목과 중복도가 높습니다. ETF 추가 매수 시 반도체 비중 과다 우려"
```

## 한계와 솔직한 표현

### 명확한 한계

1. **상위 10개 종목 외 영향**: 통상 60~80%만 커버. 나머지 20~40%는 단순 추정.
2. **편입 종목 변경**: ETF는 분기/반기마다 리밸런싱한다. 분석 시점의 편입 비중만 반영됨을 명시.
3. **레버리지/인버스 ETF**: 본 명령어는 일반 ETF용. 레버리지 ETF는 일일 추적 특성상 장기 적정가 산정이 의미 없음. 별도 안내.
4. **해외 자산 ETF**: 한국 종목 중심으로 설계되어 미국/중국 종목 ETF는 정확도 낮음.
5. **채권/원자재 ETF**: 본 명령어는 주식형 ETF 전용. 채권/원자재는 다른 방법론 필요.

### 사용자에게 명시할 점

- "ETF 적정가는 편입 종목 추정의 합이므로 오차가 누적됩니다 (각 종목 ±10% → ETF ±10~15%)"
- "편입 비중은 [날짜] 기준이며, 실제는 다를 수 있습니다"
- "운용보수가 누적되면 장기 수익률에 큰 영향. 5년 이상 보유 시 0.5% 차이도 누적 -2.5%"

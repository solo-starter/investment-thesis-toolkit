# /실적분석 [종목명] [분기]

분기 실적 발표 후 적정가를 재산정하고 히스토리 시트를 갱신하는 명령어.

## 트리거

**슬래시 명령어 형식**:
- `/실적분석 [종목명] [분기]` (예: `/실적분석 SK하이닉스 1Q26`)
- `/earnings [종목명] [분기]` (영문 별칭)

**자연어 트리거**:
- "[종목명] [분기] 실적 분석해줘"
- "[종목명] 실적 반영"
- "[종목명] 이번 분기 실적 어땠어?"
- "방금 발표된 [종목명] 실적 어떻게 봐?"

## 작업 흐름

### Step 1: 발표된 실적 데이터 수집
검색 쿼리: `[종목명] [분기] 실적 영업이익 매출 컨센서스`

수집 항목:
- 매출액 (yoy %, qoq %)
- 영업이익 (yoy %, qoq %)
- 순이익 / EPS
- 컨센서스 대비 서프라이즈/미스 (%)
- 가이던스 (다음 분기 또는 연간) 변경 여부

### Step 2: 컨센서스 변화 추적
- 12M Forward EPS 변경 (실적 발표 전 vs 후)
- 12M Forward 컨센서스 목표주가 변경
- 매수/매도 애널리스트 비율 변화
- 주요 증권사 목표가 상향/하향 명단

### Step 3: 적정가 재산정
새 EPS/매출 데이터로 4단계 시나리오 모두 재계산:

```
변화 추적
이전 EPS: XX,XXX원 → 현재 EPS: YY,YYY원 (+Z%)
이전 기준 적정가: AAA → 현재 기준 적정가: BBB (+W%)
```

### Step 4: 시장 반응 분석
- 발표 직후 주가 변동 (%)
- 적정가 변화 vs 주가 변화 비교
- "주가는 +X% 올랐는데 적정가는 +Y%만 상승" → 과열 신호
- "주가는 -X% 떨어졌는데 적정가는 +Y% 상승" → 매수 기회

### Step 5: 분기별 히스토리 시트 자동 추가

기존 엑셀 파일이 있으면 `분기별 히스토리` 시트에 다음 한 행 추가:

| 분기 | 종목 | 현재가 | 기준 적정가 | 상승여력(%) | 컨센서스 목표가 | 메모 |

메모는 핵심 포인트 1줄로 (예: "1Q26 OP 37.6조, 컨센 +12% 상회. HBM 가이던스 강세")

### Step 6: 결과 출력

```
📊 [종목명] [분기] 실적 분석 결과

✅ 발표 실적 (vs 컨센서스)
- 매출: XX조 (+X% YoY, 컨센 ±Y%)
- 영업이익: XX조 (+X% YoY, 컨센 ±Y%) ⭐
- 순이익: XX조 (+X% YoY)
- EPS: XX,XXX원

📈 컨센서스 변화
- 12M Fwd EPS: XX → YY (+/- Z%)
- 평균 목표주가: AAA → BBB (+/- W%)
- 매수 의견 비율: XX% → YY%

🎯 적정가 재산정
| 시나리오 | 이전 적정가 | 신규 적정가 | 변화 |
| 보수    | XXX        | YYY        | +Z%  |
| 기준    | XXX        | YYY        | +Z%  |
| 낙관    | XXX        | YYY        | +Z%  |

💡 시장 반응 vs 적정가 괴리
- 주가 변화: +X% (발표 후)
- 적정가 변화: +Y%
- 판단: 과열 / 합리적 / 저평가 기회

📌 다음 점검 포인트
- [차기 실적 발표일]: YYYY-MM-DD
- 핵심 관전 포인트: [...]

✅ 분기별 히스토리 시트에 자동 추가 완료
```

## 사용 예시

```
/실적분석 SK하이닉스 1Q26
→ 영업이익 37.6조 (컨센 +12% 상회)
→ 12M Fwd EPS 350K → 380K (+8.6%)
→ 기준 적정가 245만 → 266만
→ 히스토리 시트 추가
```

## 한계

- 한국 종목의 경우 잠정실적과 확정실적 발표가 분리되어 있다. 잠정실적 기준으로 분석할지 명시한다.
- 실적 발표 직후에는 컨센서스가 아직 갱신되지 않았을 수 있다. 1~2주 후 재실행이 더 정확할 수 있음을 안내한다.

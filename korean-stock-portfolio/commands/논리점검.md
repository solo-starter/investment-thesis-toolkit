# /논리점검 [종목명]

투자 논리(thesis)의 유효성을 분기별 히스토리 데이터로 추적·점검한다.

## 트리거

**슬래시 명령어 형식**:
- `/논리점검 [종목명]`
- `/thesis [종목명]` (영문 별칭)

**자연어 트리거**:
- "[종목명] 투자 논리 점검"
- "[종목명] 보유할까 매도할까?"
- "[종목명] 처음 샀을 때랑 지금 어떻게 달라?"
- "[종목명] 매수/보유 판단"

## 작업 흐름

### Step 1: 분기별 히스토리 데이터 추출

`분기별 히스토리` 시트에서 해당 종목의 모든 분기 데이터 추출:
- 분기 시점별: 현재가, 기준 적정가, 상승여력, 컨센서스 목표가
- 메모 누적

**최소 2개 분기 데이터가 있어야 추세 분석 가능**. 1개 분기만 있으면 안내 메시지와 함께 `/적정가산정` 추천.

### Step 2: 추세 분석

#### 적정가 추세
- 적정가의 분기별 변화 (상향/하향/유지)
- 변화율 (%): 분기 대비, 누적
- 추세 강도: 강한 상향 / 점진 상향 / 횡보 / 하향

#### 시장가 vs 적정가 괴리 추세
- 각 분기별 (시장가 - 적정가) / 적정가
- 괴리 추세: 축소 / 확대 / 일정

#### 컨센서스 변화 추세
- 컨센서스 목표가의 분기별 변화
- 매수/매도 의견 비율 변화

### Step 3: 투자 논리 유효성 점검

논리를 4가지 측면에서 점검:

**A) 펀더멘털 강화 신호** (논리 ✅)
- 적정가가 분기마다 상향 중
- 영업이익/EPS 컨센서스 지속 상향
- 매출 가이던스 상회 추세

**B) 펀더멘털 약화 신호** (논리 ❌)
- 적정가 하향 추세
- 컨센서스 목표가 하향
- 마진 압박 신호 (원자재, 환율, 경쟁)

**C) 시장 평가 변화**
- 적정가는 그대로인데 시장가만 폭등 → 과열 신호
- 적정가 상향 < 시장가 상승 → 주의
- 적정가 상향 > 시장가 상승 → 추가 매수 기회

**D) 새로운 리스크/기회**
- 분기별 메모에서 누적된 리스크 항목
- 새로 부각된 기회 (예: AI 재평가 진행 중)

### Step 4: 매수/보유/매도 판단 가이드

다음 매트릭스로 판단:

| 적정가 추세 | 괴리 상태 | 판단 |
|------------|----------|------|
| 강한 상향 | 시장가 < 보수 적정가 | 🟢 강력 매수 |
| 강한 상향 | 시장가 ≈ 기준 적정가 | 🟢 매수/보유 |
| 강한 상향 | 시장가 > 낙관 적정가 | 🟡 보유 (추가 매수 자제) |
| 점진 상향 | 시장가 < 기준 적정가 | 🟢 매수 |
| 점진 상향 | 시장가 ≈ 낙관 적정가 | 🟡 보유 |
| 횡보 | 시장가 < 기준 적정가 | 🟡 보유 |
| 횡보 | 시장가 > 낙관 적정가 | 🔴 분할 매도 검토 |
| 하향 | 시장가 ≥ 기준 적정가 | 🔴 매도 검토 |
| 하향 | 시장가 < 보수 적정가 | 🟡 보유 (회복 기다리기) |

### Step 5: 결과 출력

```
🔍 [종목명] 투자 논리 점검

📈 적정가 추세 (최근 N분기)
| 분기 | 시장가 | 기준 적정가 | 상승여력 | 코멘트 |
| 1Q26 | XXX,XXX | YYY,YYY | +X% | ... |
| 2Q26 | XXX,XXX | YYY,YYY | +X% | ... |
| 3Q26 | XXX,XXX | YYY,YYY | +X% | ... |

추세: 🟢 강한 상향 / 🟡 점진 상향 / ⚪ 횡보 / 🔴 하향
누적 변화: 적정가 +X%, 시장가 +Y%

✅ 논리 강화 요인
- [분기별 메모에서 추출]
- ...

❌ 논리 약화 요인
- [있다면 명시]

🆕 새로운 변수
- [분기별 메모에 등장한 새 키워드]

💡 판단 가이드
현재 상태: [매트릭스 적용 결과]
권장: 🟢 매수 / 🟡 보유 / 🔴 매도

근거:
- [구체적 이유 1]
- [구체적 이유 2]

⚠️ 주의사항
- 분기별 히스토리는 N개 분기만 누적되어 있어 장기 추세 판단에는 한계
- 거시 환경 변화 (금리, 환율) 영향 별도 점검 필요
- 본 추정은 투자 권유가 아니며, 최종 판단은 본인 책임
```

## 사용 예시

```
/논리점검 두산에너빌리티

→ 적정가 추세 (3분기): 12만 → 14만 → 18만 (강한 상향)
→ 시장가 추세: 11만 → 13만 → 14만 (적정가 상승률 < 시장가 상승률보다 큼)
→ 논리 강화: SMR 상용화, 가스터빈 호황
→ 판단: 🟢 매수/보유 가능 (적정가가 시장가를 끌고 있음)
```

## 한계

- 1~2분기 데이터로는 추세 판단이 어렵다 (최소 3분기 권장)
- 단기 사이클 vs 장기 구조적 변화 구분이 어렵다
- 논리 점검은 정량 데이터 + 정성 판단의 결합이라 100% 자동화는 불가
